# Security Policy

## Reporting a vulnerability
Please report security issues **privately** instead of opening a public issue:

- Use GitHub's private vulnerability reporting (repo **Security → Report a vulnerability**), or
- Contact the maintainer via the GitHub profile: https://github.com/s0912758806p

Please include reproduction steps and the affected version (see the version badge / `plugin.json`).

## Scope & expectations
- This project is provided **"as is"** under the MIT License, with no warranty.
- Every workflow output is a **DRAFT for human review** — never an authoritative or controlled record.
- Do **not** feed secrets, credentials, or regulated/proprietary data into the examples or inputs.
- The hooks are **project-scoped**: they only act on a project that has adopted `agentic-sop-kit/`, and no-op elsewhere.

## Supported versions
The latest published version on the `main` branch is supported.
